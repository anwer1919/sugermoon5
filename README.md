# 🧁 SugerMoon

Enterprise e-commerce platform for luxury desserts.

## 🛠️ Tech Stack
- **Frontend**: Next.js 15, React, TypeScript, Tailwind CSS
- **Backend**: NestJS, Prisma, PostgreSQL
- **Auth**: JWT + 2FA
- **Payments**: Stripe, Apple Pay, COD
